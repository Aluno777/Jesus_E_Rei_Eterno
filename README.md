# RedShield PTaaS v2.5 — Pentest as a Service

Plataforma de pentest automatizado construída em Django, rodando em Docker.

# RedShield PTaaS v2.5 — Vídeo demonstrativo
https://youtu.be/ZhV9LVRdaMI

## Início Rápido

```bash
docker compose build
docker compose up -d
```
Acesse: **http://localhost:8000**

## Manual Completo
Abra o arquivo `MANUAL.html` no navegador para o guia completo de instalação no Windows, uso e referência da API.
